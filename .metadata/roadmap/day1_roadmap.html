11:00 — 11:30 AM
Git Workflow Practice

Everyone practices the PR workflow on their assigned repository. This ensures nobody reaches Day 3 asking how to create a PR.

develop ↓ feature branch (e.g., chore/AUX-3-backend-setup) ↓ change ↓ commit (e.g., chore: initialize Spring Boot project) ↓ push ↓ PR (Title: [AUX-3] Initialize Spring Boot Backend) ↓ review ↓ merge
